## [1.0.0] - 2026-01-24

### 新增

- 🎉 首次发布
- ✨ 基于 AST 的静态代码分析引擎
- 🛡️ 6个核心安全检测规则：
  - SQL001: SQL注入检测
  - CMD001: 命令注入检测
  - SEC001: 硬编码凭据检测
  - DNG001: 危险函数检测
  - PTH001: 路径遍历检测
  - XSS001: XSS漏洞检测
- 📊 多格式报告输出：
  - 纯文本格式
  - Markdown 格式
  - JSON 格式
  - HTML 格式（带样式）
- 🖥️ 命令行接口 (CLI)
- 📁 目录和文件扫描
- 🔧 可扩展的规则系统
- ✅ 完整的单元测试套件 (21个测试用例)
- 📖 项目文档

### 技术特性

- 零外部依赖（仅使用Python标准库）
- 支持 Python 3.8+
- 插件化规则架构
- 装饰器规则注册机制

## 版本说明

### 版本号格式

`主版本号.次版本号.修订号`

- **主版本号**: 不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 变更类型

- `新增` - 新功能
- `变更` - 对现有功能的变更
- `弃用` - 即将移除的功能
- `移除` - 已移除的功能
- `修复` - 问题修复
- `安全` - 安全相关修复
